# JPFR Website — GitHub Pages

## Install
1. Open the repository used by `likelandi.github.io`.
2. Back up the current files.
3. Upload/replace these three files in the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
4. Commit the changes to the branch configured for GitHub Pages (usually `main`).
5. Wait for GitHub Pages deployment and refresh the site.

## Important
- No trackers, analytics, cookies or third-party personal data are included.
- The only external dependency is Google Fonts. To remove it, delete the font links in `index.html`; the site will fall back to system fonts.
- The email contact is `jprebosio@proton.me`.
- The site is bilingual (English/Spanish).
